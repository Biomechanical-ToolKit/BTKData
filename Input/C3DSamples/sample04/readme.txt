bodybuilder.c3d   A single large c3d file generated using
                  Oxford Metrics BodyBuilder software.
                  Stored in DEC-REAL format.

sub_labels.c3d    A file using subject labels for the point
                  data.  This is stored in DEC-INT format and
                  some of the calculated POINT variables show
                  a loss of resolution as the data values
                  approach the POINT:SCALE value.
                  Stored in DEC-INT format.
